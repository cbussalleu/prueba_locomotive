Introduction

Locomotive Scroll is a javascript library for creating smooth scrolling effects on web pages. It is lightweight, flexible, and easy to use, making it a popular choice for designers and developers who want to add a touch of interactivity to their websites.

Features

Easy to Use

Locomotive Scroll is designed to be easy to use for both novice and experienced web developers.

Flexible

Locomotive Scroll can be customized to fit any website design and layout.

Lightweight

Locomotive Scroll is a lightweight library that won't slow down your website's loading times.
